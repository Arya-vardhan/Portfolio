# **App Name**: Arya's Cyberfolio

## Core Features:

- Animated Hero Section: Display a full-screen hero section with a 3D element and animated dynamic text.
- About Me Section: Showcase education and experience in a modern card layout.
- Animated Skills Section: Interactive radial charts or progress bars to display skills.
- Experience Timeline: Animated cards that scroll into view showing work history at Apex Group and Vishnuspire.
- Project Showcase: Interactive cards for personal projects like the Reverse Vending Machine and portfolio.
- SEO Optimization Tool: Use provided keywords for SEO and LLM to incorporate new, relevant keywords.
- Contact Form with Validation: Allow users to send emails directly, with integrated validation and dark glass morphism.

## Style Guidelines:

- Primary color: Neon purple (#9400D3) for a futuristic tech feel.
- Background color: Dark gray (#222222) to enhance neon accents.
- Accent color: Neon blue (#00FFFF) to highlight interactive elements.
- Body and headline font: 'Space Grotesk' sans-serif for headlines and 'Inter' for body, to maintain a contemporary and clean aesthetic. Note: currently only Google Fonts are supported.
- Minimalist icons for hobbies and skills, emphasizing a tech-inspired look.
- Dark glassmorphism for card backgrounds to create depth and a modern aesthetic.
- Subtle parallax effects and scroll-triggered animations for enhanced interactivity.
